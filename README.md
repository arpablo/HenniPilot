# HenniPublish — Schwarzes Gold

Eine interaktive Wissensplattform zur Geschichte der Ölindustrie im Nahen Osten, gebaut mit Nuxt 4 und Nuxt UI.

## Tech Stack

- **Nuxt 4** mit Vue 3 und TypeScript
- **Nuxt Content** — Markdown-basiertes Content-Management mit MDC-Syntax
- **Nuxt UI** — Komponentenbibliothek mit Tailwind CSS 4
- **Leaflet** — Interaktive Karten
- **Mermaid** — Diagramme und Visualisierungen

## Features

- **Publikationen mit Inhaltsverzeichnis** — Mehrteilige Publikationen mit navigierbarer Kapitelstruktur
- **Link-Preview Drawer** — Vorschau verlinkter Inhalte in einem Seitenpanel
- **Interaktive Karten** — Leaflet-basierte Karten mit Markern und Popups
- **14+ Content-Collections** — Kapitel, Personen, Organisationen, Ereignisse, Länder, Städte, Orte, Essays, Begriffe, Notizen u.v.m.
- **Eigene Prose-Komponenten** — Angepasstes Rendering für Links und Bilder
- **Volltextsuche** — Suche über alle Inhalte
- **Dark Mode** — Automatische Erkennung und manueller Toggle
- **Responsive Design** — Optimiert für Desktop und Mobile

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

Startet den Entwicklungsserver auf `http://localhost:3000`.

## Production

```bash
pnpm build
pnpm preview
```

## Linting & Typechecking

```bash
pnpm lint
pnpm typecheck
```
